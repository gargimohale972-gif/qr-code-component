# QR Code Component

A simple and elegant **QR Code card component** built with HTML and CSS.  

This project is a **perfect first challenge** if you're new to HTML and CSS. The card layout is fixed, so it doesn't shift when resized, making it ideal for beginners who haven't learned responsive layouts yet.  

## Features

- Minimalistic QR code card design
- Fully built with **HTML & CSS**
- Beginner-friendly: no frameworks or libraries required
- Great first project for learning layout and styling basics

## Installation / Usage

1. Clone the repository or download the files.
2. Open `index.html` in your browser to view the component.
